# **App Name**: Verity Forecaster

## Core Features:

- Portfolio Dashboard: Display consolidated NAV projection, net cashflow timeline, and other key metrics with real-time updates.
- Fund Management: Enable users to add, edit, duplicate, and remove funds, updating key parameters like commitment, vintage year, and fee assumptions.
- Statement Extraction Tool: Automatically extract capital calls, distributions, NAV values, and other relevant data from uploaded PDF statements using OCR and intelligent data parsing.
- Cashflow Forecast Engine: Generate capital call, distribution, and NAV projections, supporting monthly and quarterly views by intelligently blending historical data with a user-customizable forecasting tool.
- J-Curve Customization: Allow users to adjust deployment pacing, distribution timing, and NAV ramp curves based on reusable templates for various asset classes.
- Scenario Simulation Manager: Enable the creation and comparison of different scenarios (e.g., base case, fast/slow deployment, downside vintage) with instant forecast updates.
- Liquidity Planning View: Visualize funding needs, liquidity gaps, and capital call spikes over time, with options to export data tables for further analysis.

## Style Guidelines:

- Primary color: Deep blue (#1A438A) to evoke trust and professionalism, suitable for institutional finance.
- Background color: Light blue-gray (#E5E9F2), offering a clean and calming backdrop.
- Accent color: Soft orange (#D97706) for highlighting key metrics and interactive elements. It is close to yellow on the color wheel and has similar brightness.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for both headlines and body text
- Use clear, professional icons to represent different asset classes and financial metrics.
- Employ a clean, modern layout with interactive charts and analytics tables, minimizing clutter for a professional finance dashboard style.
- Incorporate subtle animations for scenario changes and data updates to enhance user experience.
